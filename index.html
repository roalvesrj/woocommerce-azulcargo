Integration between the AzulCargo and WooCommerce

Descrição

Utilize os métodos de entrega e serviços da Azul Cargo com a sua loja WooCommerce.

O WooCommerce deve estar instalado e ativo.

Azul Cargo é um método de entrega brasileiro.

Este plugin foi desenvolvido sem nenhum incentivo da Azul Cargo. Nenhum dos desenvolvedores deste plugin possuem vínculos com esta empresa. Este software é gratuito e não está associado a AzulCargo. AzulCargo é uma marca registrada da Azul - Linhas Aéreas Brasileiras. Este plugin não está afiliado a Azul - Linhas Aéreas Brasileiras e, portanto, não é um produto oficial da AzulCargo. Note que este plugin foi feito baseado em cUrl, visto que a transportadora não disponibiliza documentação para integração API.
